# Darcula.nvim

A Darcula theme, for Neovim.  
Work in progress.

## Installation

For lazy.nvim users

```lua
{
    "tallestlegacy/darcula.nvim",
    dependencies = "rktjmp/lush.nvim",
    lazy = false,
}
```

For packer, i guess

```lua
{
    "tallestlegacy/darcula.nvim",
    requires = "rktjmp/lush.nvim",
    lazy = false,
}
```

And vim-plug

```vim
Plug "rktjmp/lush.nvim"
Plug "tallestlegacy/darcula.nvim"
```
