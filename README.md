# Darcula.nvim

A Darcula theme, for Neovim.  
Work in progress.

## Installation

For lazy.nvim users

```lua
{ "tallestlegacy/darcula.nvim" }
```

For packer, i guess

```lua
{ "tallestlegacy/darcula.nvim" }
```

And vim-plug

```vim
Plug "tallestlegacy/darcula.nvim"
```
